# Sample Coverage Questions

These examples can be used to test the notebook after the required data files and API key are configured.

```text
Is CPT 84443 covered for ICD-10 E03.9 on 2026-01-01?
```

```text
Is CPT 82306 covered for ICD-10 Z13.21 on 2026-01-01?
```

```text
Is CPT 87086 covered for ICD-10 A02.1 on 2026-01-01?
```

```text
Is CPT 80050 covered for ICD-10 Z00.00 on 2026-01-01?
```

Expected answer fields may include:

- Parsed CPT code
- Parsed ICD-10 code
- Coverage decision
- Relevant NCD
- Supporting evidence
- Confidence score
- Missing or insufficient information
